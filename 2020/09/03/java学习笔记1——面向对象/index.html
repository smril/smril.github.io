面向对象

抽象类

四边形可以是一个具体的类吗？平行四边形、矩形、梯形这些都可以是四边形的具体类，那么我们可以把四边形定义为一个抽象类，这个类的意义就是被继承，他里面的所有方法都没有任何代码，只有方法名和参数，他的意义也只是被重写。

定义一个抽象类的关键字是abstract，

这样的类的唯一意义就是被继承，而且继承自他的子类必须重写里面的所有用abstract关键字的方法。
